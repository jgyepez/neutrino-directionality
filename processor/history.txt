Soon to implemented PDG code filtering. Main Python file used for processing and visualized RATPAC2 data is now called 'main.py'. A new part of the code is developing a new mathematical construct I call the Continuous Frobenius Norm of the Difference (CFND). The CFND is defined as the double integral of the square of the difference between two 2D normal distributions. The integral is then squared rooted to preserve units and align with the Frobenius Norm. Code also now has discrete simulation of FND compared to both the absolute value of sine fit and CFND. Notice that for small offset of distribution from center, the CFND and FND are for all practical purposes approximately equal to the absolute value of sine. This seems to be a powerful discovery.
